# Drug Response Prediction from Cancer Cell Line Transcriptomes

Predict compound response (viability / AUC / LFC) from baseline gene expression profiles.

This repo is designed as a **portfolio-quality** project:
- strong reproducibility (pinned environment)
- leak-resistant evaluation (proper splits)
- interpretable baselines (Elastic Net)
- extensible to tree models / neural nets

## 1) Biological question
Can baseline transcriptomic profiles of cancer cell lines predict **drug sensitivity** to small-molecule compounds?

## 2) Data sources (public)
This project is set up to use **DepMap**:
- **CCLE/DepMap gene expression** (RNA-seq expression per cell line)
- **PRISM Repurposing** drug response (compound viability readouts per cell line)

Download page: DepMap portal → Downloads.

> Note: You must follow DepMap terms/attribution when using the data.

## 4) Quick start
### Create the environment
```bash
conda env create -f environment.yml
conda activate drug_response_env
```

### Put raw files into `data/raw/`
See `scripts/00_validate_inputs.py` for exact expected filenames.

### Build dataset + train baseline
```bash
python scripts/00_validate_inputs.py
python scripts/01_build_feature_matrix.py
python scripts/02_train_elasticnet.py
python scripts/03_evaluate_and_interpret.py
```

## 5) Outputs
- `data/processed/X_expression.parquet` : expression feature matrix (cell lines × genes)
- `data/processed/y_drug_response.parquet` : response matrix (cell lines × compounds)
- `results/metrics/*.json` : evaluation metrics per compound
- `results/figures/*.png` : performance & feature-importance plots
